DESI Data Inspector

Example URLs

Search form

{{ base_url }}/search

See what targets from the loa production are within 50 arcsec of RA,dec=(150,20) degrees:

{{ base_url }}/loa/targets/radec/150,20,50

Download that table as a FITS file, or as json/csv/ascii:

{{ base_url }}/loa/targets/radec/150,20,50?format=fits
{{ base_url }}/loa/targets/radec/150,20,50?format=json
{{ base_url }}/loa/targets/radec/150,20,50?format=csv
{{ base_url }}/loa/targets/radec/150,20,50?format=ascii
(for symmetry with these URLs, the original table view can also be specified with "?format=html")

Interactively view the spectra:

{{ base_url }}/loa/spectra/radec/150,20,50

Download those spectra as a FITS file, using the same format as DESI coadded spectra:

{{ base_url }}/loa/spectra/radec/150,20,50?format=fits

Instead of an RA,DEC,RADIUS search, you can also search by TARGETIDs:

{{ base_url }}/loa/targets/39628261201809937,39628261205999711
Note: All of the same "?format=html|fits|json|csv|ascii" options work for this TARGETID-based view as well.

View the spectra for those TARGETIDs:

{{ base_url }}/loa/spectra/39628261201809937,39628261205999711

Download a FITS file with the spectra for those TARGETIDs:

{{ base_url }}/loa/spectra/39628261201809937,39628261205999711?format=fits

By default, the results are for healpix-based spectra which coadd data for the same TARGETID when observed on multiple tiles. To get tile-based spectra instead:

{{ base_url }}/loa/targets/tiles/radec/150,20,50
{{ base_url }}/loa/spectra/tiles/radec/150,20,50
{{ base_url }}/loa/targets/tiles/39628261201809937,39628261205999711
{{ base_url }}/loa/spectra/tiles/39628261201809937,39628261205999711

Although healpix is default, it can also be explicitly specified for symmetry with the tile-based URLs:

{{ base_url }}/loa/targets/healpix/radec/150,20,50
{{ base_url }}/loa/spectra/healpix/radec/150,20,50
{{ base_url }}/loa/targets/healpix/39628261201809937,39628261205999711
{{ base_url }}/loa/spectra/healpix/39628261201809937,39628261205999711
