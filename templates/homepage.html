The DESI Data Inspector provides interactive browsing of DESI data and programatic URLs for directly downloading subsets of the data.

Use the Search page, or generate URLs like 

RELEASE/ targets|spectra / [tiles|healpix] / TARGETIDs|radec/RA,DEC,RADIUS|TILEID/FIBERs [?format=html|fits|json|csv|ascii]
where
  • RELEASE is the Data Release or Spectrosopic production name, e.g. edr, dr1, fuji, iron.
  • targets or spectra selects between getting a table of targets or their spectra.
  • tiles or healpix selects between tile-based or cross-tile healpix-based data (default healpix).
  • TARGETIDs is a comma separated list of integer TARGETIDs.
  • radec/RA,DEC,RADIUS performs a cone search of RADIUS arcsec around RA,DEC (degrees).
  • TILEID/FIBERs loads the comma-separated list of FIBERs for tile TILEID.
  • Optional format parameters control whether the results are renders as a webpage or provided as a format to download (default html). html and fits are valid formats for both spectra or targets tables; the other formats apply only to targets tables.
See the Examples pages for details.
