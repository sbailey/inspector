The DESI Data Inspector provides interactive browsing of DESI data and programatic URLs for directly downloading subsets of the data. Use the Search page, or generate URLs like {% include "url_structure.html" %}

For example, Data Release 1 (DR1) targets within 60 arcsec of RA,dec=(210,5) degrees while filtering to those identified as galaxies with redshift z<0.1:

{{ root_url }}/dr1/targets/radec/210,5,60?SPECTYPE=GALAXY&Z=lt:0.1

Or download the spectra for those targets in FITS format:

{{ root_url }}/dr1/spectra/radec/210,5,60?SPECTYPE=GALAXY&Z=lt:0.1&format=fits

See the Examples page for more URL examples and details.
